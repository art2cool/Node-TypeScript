# Node + TypeScript example.

### Installation

```sh
$ npm install
$ npm install gulp -g
```
### Run

start server for first-tab:
```sh
$ gulp
$ npm start
```

### Notes

```
GET 'http://localhost:3000/api/v1/heroes' -get all heroes
GET 'http://localhost:3000/api/v1/heroes/1' -get hero with id 1
```
